## What is Cadre?

Cadre is a **managed infrastructure platform for AI agent teams**. We take [Paperclip](https://github.com/paperclipai/paperclip) — the open-source orchestration layer for zero-human companies — and turn it into a production-ready, fully isolated service running on **your own server**.

No Docker setup. No `.env` debugging. No token waste. You fill out a form, pick your hosting provider, and log in to a running AI company with departments, roles, budgets, and monitoring — ready to delegate.

## Why Cadre?

- **Zero-setup deployment** — Automated server provisioning on Hetzner, Netcup, or DigitalOcean
- **Pre-optimized prompts** — Up to 60% fewer tokens per task compared to vanilla Paperclip
- **Gemini CLI ready** — Subscription-based LLM access out of the box, no per-token billing surprises
- **Full isolation** — Every customer gets their own hardened server (SSH, firewall, auto-updates)
- **Department RBAC** — Granular permissions per business unit and agent role
- **Real-time monitoring** — Agent heartbeats, cost tracking, and status dashboards
- **GDPR compliant** — Your data stays on your server, hosted in the EU

## Tech Stack

Cadre builds on top of Paperclip (MIT) and extends it with:

- **Infrastructure**: Docker Compose · Traefik · Hetzner Cloud API · Ansible
- **Backend**: Node.js · Gemini CLI
- **Monitoring**: Custom heartbeat system · cost-per-agent tracking
- **Security**: Fail2ban · UFW · SSH hardening · automated OS updates
